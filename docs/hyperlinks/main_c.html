Contiene la funcion 

 int main ()

La funcion main implementa una suerte de maquina de estado

Su funcionamiento se basa en el pasaje de un objeto estado comun a todas las "pantallas" del programa, que estas utilizan para decidir a cual pantalla pasar despues

este objeto es de tipo 

 estado_t
y se pasa por puntero de una funcion a otra, cada funcion puede leer y mutar el objeto estado.
